# TerminalSetup

### Setup using bash

```
cd TermainalSetup
bash setup.sh
```

## Recipe

### 1. Navigate into folder

```
cd TermainalSetup
```

### 2. Unzip Dependencies

```
open powerlevel10k.zip
open zsh-autocomplete.zip
```

### 3. Create folder for plugins

```
mkdir ~/plugins
```

### 4. Move files

```
mv powerlevel10k ~/plugins
mv zsh-autocomplete ~/plugins
mv .vimrc ~/.vimrc
mv .zshrc ~/.zshrc
```
